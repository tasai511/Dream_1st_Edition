# ドリーム1号 素振り記録アプリ

素振り練習用デバイス「ドリーム1号」の記録用PWAです。HTML / CSS / JavaScriptのみで構成され、アプリ本体は `app/` フォルダに入っています。

## できること

- 名前とバットを複数登録
- 日付、名前、バット、素振り回数、平均スコア、ベストスコアを記録
- 同じ日付・名前・バットの記録は自動で合算
- 平均スコアは回数による加重平均で計算
- ベストスコアは最大値を採用
- 最近の入力履歴から編集・削除
- 期間、名前、バットで絞り込んだグラフ表示
- CSVエクスポート / インポート
- オフライン動作

## GitHub Pagesで公開する方法

1. このリポジトリをGitHubへpushします。
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
5. `Branch` で公開したいブランチと `/root` を選び、保存します。
6. 表示されたURLを開くと、自動で `app/` のアプリが開きます。

アプリを直接開く場合は、GitHub PagesのURL末尾に `/app/` を付けてください。

## PWAとしてホーム画面に追加する方法

### iPhone / iPad

1. SafariでGitHub PagesのURLを開きます。
2. 共有ボタンを押します。
3. `ホーム画面に追加` を選びます。
4. 名前を確認して追加します。

### Android

1. ChromeでGitHub PagesのURLを開きます。
2. メニューから `ホーム画面に追加` または `アプリをインストール` を選びます。
3. 確認して追加します。

## CSVエクスポート / インポート

### エクスポート

`CSVエクスポート` ボタンを押すと、全記録をCSVファイルとして保存します。CSVには以下の列が含まれます。

- 名前
- バット
- 日付
- 回数
- 平均
- ベスト

### インポート

`CSVインポート` からCSVファイルを選ぶと、記録を復元できます。既存データと同じ日付・名前・バットの行は合算されます。不正な行はスキップされ、読み込み後にスキップ件数が表示されます。

## データ保存について

データはすべて端末ローカルのIndexedDBに保存されます。サーバー保存、ログイン、クラウド同期はありません。

端末を変更する場合やブラウザのデータを消す場合は、事前にCSVエクスポートでバックアップしてください。新しい端末ではCSVインポートで移行できます。
